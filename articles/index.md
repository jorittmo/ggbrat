# Articles

### Using atlases

- [Plotting ggbrat
  atlases](https://jorittmo.github.io/ggbrat/articles/plotting-atlases.md):

  Actually use the atlases. Join brain data, add shading and atlas
  overlays, and create animations.

- [Resources and
  caching](https://jorittmo.github.io/ggbrat/articles/resources-and-caching.md):

  Find, download, load, and manage ggbrat atlases and source files.

### Building atlases

- [Building surface
  atlases](https://jorittmo.github.io/ggbrat/articles/building-surface-atlases.md):

  Project cortical annotations and labelled meshes into custom 2D ggplot
  atlases.

- [Building volumetric
  atlases](https://jorittmo.github.io/ggbrat/articles/building-volume-atlases.md):

  Turn axial, sagittal, and coronal NIfTI slices into sf polygons.

- [Building SVG
  atlases](https://jorittmo.github.io/ggbrat/articles/building-svg-atlases.md):

  Author labelled vector templates and import them as sf atlases.

### Data sources

- [Using
  TemplateFlow](https://jorittmo.github.io/ggbrat/articles/templateflow.md):

  Download TemplateFlow files and pass them to ggbrat’s existing atlas
  builders.
